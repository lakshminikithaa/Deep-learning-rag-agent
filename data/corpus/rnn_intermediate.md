# Recurrent Neural Networks

## Processing Sequential Data with Hidden States
When processing data where temporal sequence matters, such as text, speech, or time-series readings, classical models fall short because they assume all inputs are independent of one another. To address this, specialized architectures maintain an internal memory, generally known as a hidden state. As the model ingests a sequence step-by-step, the hidden state is continuously updated based on both the current input token and the hidden state from the immediately preceding time step. This recurrent loop allows information from earlier in the sequence to persist and influence the processing of later elements. Through this mechanism, the system can understand context, grammar, or trends over time. Because the exact same weights are applied at every time step to process the input and update the state, the model can flexibly handle sequences of variable lengths, making it highly suitable for tasks like natural language translation or stock price prediction.

## The Vanishing Gradient Problem in Time-Series Models
Training architectures that process sequential data relies on an algorithm called Backpropagation Through Time. This involves unrolling the recurrent loop into a deep feedforward structure, where each layer represents a single time step. As gradients are calculated backwards from the final output to the early inputs, they are repeatedly multiplied by the same recurrent weight matrices. If the eigenvalues of these matrices are less than one, the gradients shrink exponentially with each backward step. This phenomenon, known as the vanishing gradient problem, prevents the early states of the sequence from receiving meaningful weight updates based on errors occurring much later in the sequence. Consequently, the model exhibits a short-term memory, successfully learning immediate dependencies but entirely failing to capture long-range contextual relationships, which is a major limitation when processing lengthy paragraphs of text or long audio recordings.

## Long Short-Term Memory and Gating Mechanisms
To overcome the severe limitations of standard memory-based sequential models, Long Short-Term Memory networks introduce a more complex internal structure centered around a cell state and three distinct gating mechanisms. The cell state acts as an information highway that runs straight down the entire sequence, with only minor linear interactions, allowing gradients to flow backwards uninterrupted. The control of what information enters or leaves this highway is managed by the forget gate, the input gate, and the output gate. The forget gate determines which irrelevant past information should be discarded from the cell state. The input gate decides which new, relevant information from the current step should be added. Finally, the output gate determines what portion of the cell state should be exposed as the hidden state for the current step. This precise, learned gating architecture allows the system to selectively maintain or clear information over arbitrarily long sequences, effectively solving the vanishing gradient issue.
