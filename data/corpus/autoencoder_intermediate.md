# Autoencoders

## Dimensionality Reduction and Latent Space
An Autoencoder is an unsupervised neural network designed to learn efficient compressions of data. It consists of two symmetric parts: an Encoder and a Decoder. The Encoder compresses the high-dimensional input data into a much lower-dimensional representation, known as the latent space or bottleneck. The Decoder then attempts to reconstruct the original input from this compressed latent representation. The network is trained to minimize the reconstruction error between the original input and the regenerated output. Because the latent space has a lower dimensionality than the input, the network is forced to learn the most salient features and discard noise, effectively performing non-linear dimensionality reduction akin to Principal Component Analysis (PCA).

## Denoising Autoencoders
A standard autoencoder might simply learn the identity function by memorizing the input if the latent space is too large, rather than extracting meaningful structure. A Denoising Autoencoder prevents this by deliberately corrupting the input data—such as adding Gaussian noise or randomly dropping inputs—before feeding it into the Encoder. However, the loss function still penalizes the difference between the Decoder's output and the original, uncorrupted input. This forces the network to learn robust, noise-invariant representations of the data and genuinely understand the underlying structure in order to "denoise" and reconstruct the original signal.

## Sparse Autoencoders
Another regularization technique to prevent identity learning is the Sparse Autoencoder. In a Sparse Autoencoder, the latent space might actually have a higher dimensionality than the input, which would normally lead to trivial memorization. However, a sparsity penalty is added to the loss function, which restricts the number of active neurons in the hidden layer at any given time. This encourages the network to learn highly specialized and localized feature detectors, similar to how the human visual cortex operates. By forcing most neurons to be inactive (sparse) for a given input, the model learns a more robust and disentangled representation of the data.
