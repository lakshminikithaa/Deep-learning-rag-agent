# Artificial Neural Networks

## The Perceptron and Linear Separability
The foundational building block of classical feedforward models is the perceptron. A single perceptron takes multiple input values, applies a specific weight to each input, sums them up, and adds a bias term. This weighted sum is then passed through a step function to determine the final output. While this simple mathematical model successfully mimics basic logical operations like AND or OR gates, it possesses a fundamental limitation: it can only solve problems that are linearly separable. For instance, the classic XOR problem cannot be solved by a single perceptron because plotting the outputs of an XOR gate on a two-dimensional plane reveals that a single straight line cannot separate the true and false outputs. To overcome this limitation and solve complex, non-linear problems, multiple perceptrons must be combined into interconnected layers, forming a deeper architecture. This multi-layered approach allows the model to learn complex decision boundaries by hierarchically combining simpler linear separations.

## Backpropagation and Weight Updates
The primary mechanism for training multi-layered models relies on an algorithm known as backpropagation. During the training process, an input is passed forward through the network layers to generate a prediction. This prediction is then compared to the ground truth label using a loss function, which quantifies the error. Backpropagation efficiently calculates the gradient of this loss function with respect to every weight in the system. It does this by applying the chain rule of calculus, starting from the final output layer and working backward to the initial input layer. Once these gradients are calculated, an optimization algorithm, most commonly gradient descent or one of its variants like Adam, uses them to update the weights. The weights are adjusted in the direction that minimizes the overall error. This iterative process of forward passes and backward parameter updates allows the architecture to progressively refine its internal representations and improve its predictive accuracy over time across many training epochs.

## Activation Functions and Network Non-Linearity
In order to learn complex patterns, a network necessitates the introduction of non-linearity through activation functions. If classical feedforward models only consisted of linear transformations—multiplying inputs by weights and adding biases—the entire multi-layered structure would collapse mathematically into a single linear transformation. Consequently, the model would only be capable of learning linearly separable data, regardless of its depth. Activation functions are applied to the output of each node before the signal is passed to the next layer, breaking this linearity. Historically, the sigmoid or hyperbolic tangent functions were popular, but they often lead to vanishing gradients during deep training. Modern architectures predominantly utilize the Rectified Linear Unit (ReLU), which outputs the input directly if it is positive, and zero otherwise. ReLU is computationally efficient and significantly mitigates the vanishing gradient issue, enabling the successful training of exceedingly deep feedforward models.
